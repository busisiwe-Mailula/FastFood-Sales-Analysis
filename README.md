# FastFood-Sales-Analysis
End-to-end data analysis project using Python, MySQL, and Power BI

##Project Overview
This project analyses fast food sales data to uncover key business insights.

#Tools used
-Python
-MySQL
-Power BI

##Project structure
-data/ -> datasets
-src/ -> data cleaning scripts
-sql/ -> SQL queries
-report/ -> Power BI dashboard

##Key Insights
-Burger generates the most revenue and is the most sold item
-Fanta is underperforming due to low demand
